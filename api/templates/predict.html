Jak działa prognoza?

Model XGBoost wytrenowany na danych z Kaggle przewiduje roczne wynagrodzenie brutto w USD na podstawie 9 cech oferty pracy.

Cechy wejściowe

  • Stanowisko — tytuł zawodowy (np. Data Scientist, ML Engineer)
  • Lata doświadczenia — całkowity staż zawodowy
  • Wykształcenie — najwyższy osiągnięty poziom edukacji
  • Branża — sektor firmy (Technology, Finance…)
  • Wielkość firmy — Small / Medium / Large
  • Lokalizacja — kraj / region
  • Praca zdalna — Yes / No / Hybrid
  • Liczba umiejętności — wartość liczbowa z CV
  • Certyfikaty — liczba zdobytych certyfikatów branżowych

REST API

Prognozę można też pobrać programatycznie:

  • POST /predict z body JSON
  • Dokumentacja: Swagger UI ↗

Ograniczenia

Model interpoluje w zakresie danych treningowych. Skrajne kombinacje (np. 0 lat + CEO) mogą dawać zawyżone lub zaniżone wyniki.
