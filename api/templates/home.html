O projekcie

System predykcji wynagrodzeń zbudowany w ramach projektu zespołowego z zakresu hurtowni danych i metod analitycznych.

Pipeline (4 kroki)

  • Dane ETL — pobranie CSV z Kaggle, oczyszczenie (silver), agregaty (gold), załadowanie do Azure SQL
  • Model — trening XGBoost, logowanie eksperymentów w MLflow, wersjonowanie artefaktów przez DVC
  • Analiza — interaktywny dashboard z wykresami Chart.js oraz formularz prognozy pensji
  • Monitoring — Evidently wykrywa drift danych, Prefect planuje cotygodniowy auto-retrain

Technologie

  • Azure Data Lake Gen2 + Azure SQL (schemat gwiazdy)
  • XGBoost, scikit-learn, MLflow, DVC
  • FastAPI, Prefect, Evidently
  • Docker / docker-compose

Uruchomienie

Skopiuj .env.example → .env, umieść CSV w input/, uruchom docker compose up --build.
