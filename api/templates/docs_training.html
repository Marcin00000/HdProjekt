Model XGBoost

Regresja na warstwie silver. Hiperparametry (learning rate, max depth, n_estimators) dobierane przez GridSearchCV i logowane w MLflow.

Metryki

  • RMSE — pierwiastek z błędu kwadratowego (niższy = lepszy)
  • MAE — średni błąd absolutny w USD
  • MAPE — procentowy błąd względny
  • R² — współczynnik determinacji (0–1, wyższy = lepszy)

MLflow

Każdy trening tworzy nowy experiment run z parametrami, metrykami i artefaktami modelu. Modele trafiają do Model Registry.

DVC

Wersjonowanie danych i modeli z zdalnym storage w Azure Blob Storage. Polecenia dvc push/pull synchronizują artefakty między środowiskami.

Artefakty

  • models/salary_model.pkl — wytrenowany model
  • models/preprocessor.pkl — pipeline sklearn
  • models/training_baseline.parquet — dane bazowe dla driftu
